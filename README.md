Intro To The Web
================
##Installation
Need to clone repository and change into directory:
```
$ git clone git@github.com:Andrew47/intro_to_the_web.git
$ cd intro_to_the_web
```

##Usage
Open `cat_html_file.html`. This displays a picture of a cat with a red border.
